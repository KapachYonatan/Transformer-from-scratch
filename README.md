# Transformer from Scratch

Implementation of a Transformer language model from scratch as part of coursework.

## Project Structure

- `code/`: Model components, training logic, and tests.
- `data/`: Input corpora and datasets.

## Training Metric

Experiments in this repository optimize and report training loss only.

## Quick Start

1. Create and activate a Python environment.
2. Install dependencies used by the codebase (primarily PyTorch).
3. Run tests from the `code/` directory:

```bash
python tests.py
```
